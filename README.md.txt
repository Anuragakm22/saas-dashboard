# SaaS Dashboard Interface (Week 1 Project)

This project is part of the *GetIntern Advanced UI & Web Engineering Cohort (8 Weeks).*  
The Week 1 sprint focuses on building a multi-panel SaaS product & analytics dashboard using pure HTML5 and CSS3 — no frameworks or libraries.

## Project Overview
The dashboard demonstrates:
- **CSS Grid Layout**: Sidebar + main content grid.
- **Flexbox Components**: KPI cards and responsive sections.
- **Responsive Design**: Works on desktop, tablet, and mobile.
- **User Form**: Add new users with name, email, and role.
- **System Status**: Server health, traffic sources, recent activity.
